# 2FA - SMS 인증번호 자동 복사

macOS에서 SMS로 수신된 인증번호를 자동으로 클립보드에 복사하는 유틸리티입니다.

## 기능

- SMS 수신 시 4~8자리 인증번호 자동 감지
- 클립보드에 자동 복사 + macOS 알림
- 60초 후 클립보드 자동 정리
- 백그라운드 실행 (Dock 아이콘 없음)

## 설치

### 다운로드

[Releases](../../releases) 페이지에서 최신 `.dmg` 파일을 다운로드합니다.

### 설치 방법

1. `.dmg` 파일을 열고 `2fa.app`을 `Applications` 폴더로 드래그
2. 최초 실행 시 Gatekeeper 경고가 나타나면:
   - `2fa.app`을 **우클릭** → **열기** 클릭
   - 경고 팝업에서 **열기** 클릭 (최초 1회만 필요)
3. **시스템 설정** → **개인정보 보호 및 보안** → **전체 디스크 접근 권한**에서 `2fa` 허용

### 로그인 시 자동 실행

1. **시스템 설정** → **일반** → **로그인 항목** 열기
2. `+` 버튼 → `Applications`에서 `2fa` 선택

## 로컬 빌드

```bash
# 빌드
bash scripts/build-dmg.sh

# 결과물: dist/2fa-0.1.0.dmg
```

## 요구사항

- macOS 12 이상
- 전체 디스크 접근 권한 (Messages 데이터베이스 읽기용)
